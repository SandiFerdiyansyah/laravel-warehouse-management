## üîç Quick Index

### Key Concepts
- **reCAPTCHA v3:** Invisible, behavioral-based, score (0-1)
- **Three Tiers:** Block (<0.3), Challenge (0.3-0.5), Allow (‚â•0.5)
- **Rate Limiting:** 5 attempts per 15 minutes per IP
- **Challenge:** 10-minute window, IP validation, encrypted credentials
- **OWASP:** 100% compliance, 10/10 principles

### File Locations
- Code: `app/Http/Controllers/Auth/LoginController.php`
- Challenge: `resources/views/auth/challenge.blade.php`
- Routes: `routes/web.php`
- Config: `.env` + `config/services.php`
- Logs: `storage/logs/laravel-YYYY-MM-DD.log`

### Commands
```bash
# View logs
tail -f storage/logs/laravel-$(date +%Y-%m-%d).log

# Cache config
php artisan config:cache

# Clear logs
> storage/logs/laravel-*.log

# Test authentication
Open: http://127.0.0.1:8000/login
```

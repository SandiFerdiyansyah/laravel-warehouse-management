## 🔍 What Happens When...

### User Logs In (Normal)
```
1. reCAPTCHA scores: 0.9 (high)
2. Score ≥ 0.5 → ALLOW
3. Normal login flow
4. Redirect to dashboard ✅
```

### User Logs In from New Device
```
1. reCAPTCHA scores: 0.4 (medium)
2. Score 0.3-0.5 → CHALLENGE
3. Challenge page displayed
4. User verifies (10-min window)
5. IP validated
6. Login completes ✅
```

### Bot Tries to Attack
```
1. reCAPTCHA scores: 0.1 (low)
2. Score < 0.3 → BLOCK
3. Alert logged
4. Rejected ❌
```

### Attacker Tries 6 Times
```
1. Attempts 1-5: Checked
2. Rate limit triggered
3. 15-minute lockout
4. Rejected ❌
```
